Nossos Princípios Éticos

1. Seus Dados são Sacerdotais

O Owngorithm é uma ferramenta de introspecção. Seus dados nunca são vendidos para terceiros ou usados para treinar modelos de publicidade externa.

2. Soberania do Algoritmo

Diferente de redes sociais que otimizam para retenção, nós otimizamos para clareza. Você é o dono dos parâmetros que definem o que é "bom" ou "ruim" no seu consumo.

3. Transparência Radical

Sempre que possível, explicamos como cada gráfico é gerado. Não há "caixa preta" proposital aqui.

4. Direito ao Esquecimento

Se você apagar sua conta, todos os snapshots de dados em nosso Firestore serão deletados permanentemente em até 24 horas.
